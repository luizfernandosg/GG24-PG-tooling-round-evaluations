# GG24 Growth Pool Proposal Evaluations

This repository contains detailed evaluations of proposals submitted to the GG24 Public Goods Tooling Development Round - Growth Pool.

## Structure

- `evaluations/` - Individual evaluation reports for each proposal
- `251001 PG Tooling GG24.md` - Main tracking document with rubric scores and personal evaluator notes
- `evaluation-process.md` - Evaluation process documentation
- `evaluation-template.md` - Template used for generating evaluations
- `review points.md` - Quick reference for evaluation criteria

Each evaluation is saved as: `evaluations/[proposal-name]-evaluation.md`

## Evaluation Process

1. Proposal screenshot shared with AI assistant
2. AI analyzes against Growth Pool rubric
3. Structured evaluation generated using template
4. Evaluator reviews and refines
5. Saved to this directory for reference

## Rubric Overview (100 points)

- **Interoperability (25 pts)** - Integration with 2+ public goods primitives
- **UX Impact (20 pts)** - Clear UX problem definition and solution
- **Product Maturity & Feasibility (15 pts)** - Realistic scope for 4-6 weeks
- **Collaboration (10 pts)** - Named partners and partnerships
- **Team Track Record (10 pts)** - Proof of past shipping
- **Measurement Plan (10 pts)** - Karma GAP and CIDS framework alignment
- **Budget & Timeline (10 pts)** - Reasonable funding request with clear milestones

## Usage

Evaluations in this repository are used to:
- Inform conviction voting decisions
- Provide feedback to builders on CharmVerse
- Discuss proposals with evaluation council
- Track evaluation progress throughout the round
