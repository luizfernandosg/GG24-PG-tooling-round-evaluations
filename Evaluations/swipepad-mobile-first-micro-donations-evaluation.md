## Key Aspects

### Integration/Development in Question
Production-ready SwipePad Web DApp on Celo Mainnet enabling gasless micro-donations to climate projects via social login (Thirdweb Wallet), integrated with Regen Eliza AI agent. Deliverables include: (1) Production-Ready SwipePad Web DApp on Celo Mainnet - browser-based, mobile-responsive DApp with social login (Google, X, etc.) using Thirdweb Wallet, gasless donations in cUSD, cEUR, and GLO via Celo's fee delegation, displays 50+ vetted SDG-aligned climate projects, integrates Regen Eliza AI agent for research, content generation, and outreach, includes optional Self Protocol attestation, accepted when publicly accessible URL live on Celo Mainnet with demo showing full user journey; (2) Publish and Implement EAS Schema for "Verified Climate Project" - reusable public EAS schema enabling ecosystem-wide adoption, accepted when schema deployed and publicly visible at EAS Scan URL, at least 10 onchain attestations issued for real vetted projects, markdown documentation published.
